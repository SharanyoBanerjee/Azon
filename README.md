# 🛒 Azon – Mini Amazon Clone

Azon is a modern e-commerce web application inspired by Amazon. It allows users to browse products, add them to cart, manage wishlist, and experience a smooth shopping interface with persistent data using localStorage.

---

## 🚀 Features

- 🔍 **Search Products**  
  Search items by name or brand in real-time.

- 🛍️ **Product Listing & Categories**  
  Filter products by categories like Electronics, Kitchen, and Sports.

- 🛒 **Cart Functionality**
  - Add to cart  
  - Increase / decrease quantity  
  - Remove items  
  - Clear cart  
  - Checkout (UI simulation)

- ❤️ **Wishlist**
  - Add/remove items from wishlist  
  - Separate wishlist page  

- ⭐ **Product Ratings**
  - Rate products using star rating system  
  - Ratings persist using localStorage  

- 🌙 **Dark Mode**
  - Toggle between Light and Dark themes  

- 👤 **User Settings**
  - Update username  
  - Preferences saved in localStorage  

- 💾 **Persistent Data**
  - Cart, wishlist, ratings, theme, and username stored in localStorage  

- 📦 **Product Details Modal**
  - View detailed product info in a popup  

- 🔔 **Toast Notifications**
  - Feedback for actions like adding to cart  

---

## 🧑‍💻 Tech Stack

- **Frontend:** React (Hooks)
- **State Management:** useState, useEffect
- **Styling:** Inline CSS
- **Storage:** Browser localStorage

- Browse products on the homepage  
- Use the search bar to find items  
- Add products to cart or wishlist  
- Adjust quantities in cart  
- Rate products  
- Toggle dark mode in navbar/settings  
- Update username in settings
